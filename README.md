## Documentation

### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requirments.txt
```

### How To Start

```shell
$python main.py
```

### Dependency

- python==3.10
- django==3.0.0
- requirments^0.10.0
